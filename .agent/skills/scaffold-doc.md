# Scaffold-Doc Skill

- **Protocol:**
  1. Include YAML frontmatter with `description` and `type` (Spec, Workflow, Log).
  2. Use H1 for titles and H2/H3 for hierarchical sections.
  3. Ensure all checklists use `- [ ]` and all tables are properly aligned.
  4. Always use standard punctuation: **never** use the "—" symbol.
